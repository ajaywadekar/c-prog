Course Content

  1. Introduction to Computers, Programming Languages, and C Language Overview
  2. Structure of a C Program, Compilation Process, and Execution Steps
  3. Basic Syntax, Tokens, and Structure of a C Program (main function, header files)
  4. Variables, Constants, Data Types, and Type Modifiers in C
  5. Input and Output Functions: printf(), scanf(), and format specifiers
  6. Operators in C – Arithmetic, Relational, Logical, Assignment, and Increment/Decrement
  7. Operators in C – Bitwise, Conditional (?:), sizeof, and Comma Operator
  8. Expressions, Operator Precedence, and Associativity
  9. Type Casting (Implicit and Explicit) and Escape Sequences
  10. Decision Making: Introduction to Control Statements
  11. if Statement – Syntax, Flowchart, and Examples
  12. if...else Statement – Syntax, Flowchart, and Examples
  13. Nested if...else and Ladder if...else – Syntax, Flowchart, and Examples
  14. switch Statement – Syntax, Flowchart, and Examples
  15. switch vs if...else – Use cases and Best Practices
  16. Conditional/Ternary Operator in Decision Making
  17. Revision of Decision Making Statements with Practice Problems
  18. Introduction to Loops – While, Do-While, and For Loops Overview
  19. while Loop – Syntax, Flowchart, and Examples
  20. do-while Loop – Syntax, Flowchart, and Examples
  21. for Loop – Syntax, Flowchart, and Examples
  22. Loop Flow Control – break and continue Statements
  23. Nested Loops – Syntax, Flowchart, and Examples
  24. Infinite Loops – Causes, Prevention, and Examples
  25. Loop Applications – Table Generation, Factorials, Sum of Series
  26. Loop Applications – Prime Number Check, Fibonacci Series
  27. Loop Optimization – Minimizing Iterations, Best Practices
  28. Common Loop Errors and Debugging Techniques
  29. Introduction to goto Statement and its Limited Use in Loops
  30. Revision of Looping Statements with Practice Problems
  31. Mixed Control Structures – Combining Loops and Decision Making
  32. Real-world Problem Solving using Loops and Decision Making
  33. Lab Exercises – Implementing Practical Problems
